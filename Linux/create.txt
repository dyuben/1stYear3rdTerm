chmod - changing the permissions of user group and other account.

. - itself
.. - parent directory

ex. 
chmod u+x file1 - user execute file

chmod u-w file1	- user clear write

chmod ug+w file1 - user group add write

chmod a+x file1 - make ugo executable

chmod a-r file1 - clear all read


ADDING USER ACCOUNTS AND GROUPS

ex. 
adduser - to add an account
sudo adduser
